Führe alle Tests aus: Unit-Tests mit Coverage und Playwright E2E-Tests.

1. Unit-Tests + Coverage ausführen:
```bash
npm test
```

2. Ergebnis anzeigen (Passed/Failed, Coverage-Zusammenfassung)

3. Playwright E2E-Tests ausführen:
```bash
npx playwright test
```

4. Ergebnis anzeigen

5. Falls irgendwelche Tests fehlschlagen: Fehler analysieren und Ursache benennen.

Hinweis: Der HTML Coverage-Report ist unter `coverage/index.html` verfügbar (`npm run coverage` generiert ihn neu).
